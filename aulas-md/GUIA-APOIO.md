# Guia de Apoio para Criação das Aulas

## Objetivo

Padronizar a criação das aulas para manter:

- consistência visual;
- clareza didática;
- compatibilidade offline.

## Estrutura sugerida por aula

1. Título da aula
2. Objetivo
3. Agenda
4. Blocos de conteúdo (tópicos)
5. Exemplo prático
6. Exercício
7. Fechamento

## Regras de conteúdo

- Linguagem direta e profissional.
- Seções curtas, com listas quando possível.
- Tabelas apenas quando agregarem clareza.
- Evitar textos longos sem quebra.

## Regras técnicas (obrigatórias)

- O HTML final deve ser um arquivo único.
- CSS e JS devem ficar embutidos no próprio HTML.
- Não usar bibliotecas remotas (ex.: Font Awesome CDN).
- Ícones devem ser locais (Unicode/SVG embutido).

## Checklist antes de fechar a aula

- [ ] Menu lateral atualizado.
- [ ] Subtópicos da aula funcionando.
- [ ] Aulas futuras bloqueadas (quando ainda sem conteúdo).
- [ ] Layout testado em desktop e celular.
- [ ] Tema claro/escuro consistente.
- [ ] Sem dependências de internet.
